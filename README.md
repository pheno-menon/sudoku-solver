# Sudoku Solver

A web-based Sudoku solver built with JavaScript and Tailwind CSS

Live demo link: https://pheno-menon.github.io/sudoku-solver/ 

## Features
- Interactive 9×9 grid
- Solve and clear actions
- Light/Dark mode toggle

## Tech Stack
- Vite
- Tailwind CSS
- JavaScript

## Setup
```bash
npm install
npm run dev
